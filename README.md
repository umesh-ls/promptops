# PromptOps — AI learning game

A gamified season of missions teaching a 13-year-old how AI works and how to use Claude well.

## Hosting (GitHub Pages)
1. Upload these files to a public GitHub repo (drag & drop on github.com works).
2. Repo → Settings → Pages → Source: "Deploy from a branch" → Branch: `main`, folder `/ (root)` → Save.
3. After ~1 minute the game is live at `https://<username>.github.io/<repo>/`.

## One-time setup on the player's laptop
1. Open the game link.
2. Click **PARENT** (top right) and enter the parent PIN.
3. Paste a Claude API key (create at console.anthropic.com) — powers the live missions.
4. Enter your duel prompt for the boss fight and set the season prize.

Progress, the key, and duel prompts are stored in that browser only — always play on the same laptop and browser.
